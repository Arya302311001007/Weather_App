#  My Awesome Weather App

A modern, responsive weather application built using React, JavaScript, HTML, and CSS. Get real-time weather updates and forecasts for any location worldwide.

##  Features

* **Global Location Search:** Find the weather for any city.
* **Real-time Data:** Displays current temperature, humidity, and wind speed.
* **Forecast View:** Provides a short-term weather outlook.
* **Intuitive UI:** Clean and responsive design.

##  Tech Stack

* **Frontend:** React.js
* **Language:** JavaScript (ES6+)
* **Markup:** HTML5
* **Styling:** CSS3
* **API:** [OpenWeatherMap]

##  Getting Started

Follow these steps to set up the project locally.

### Prerequisites

* Node.js (LTS recommended)
* npm or yarn

### Installation

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/YourUsername/weather-app.git](https://github.com/Arya302311001007/Weather_App.git)
    cd weather-app
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    # or yarn install
    ```
3.  **API Key Setup:**
    Create a file named `.env` in the root directory and add your weather API key:
    ```
    REACT_APP_API_KEY=YOUR_SECRET_API_KEY_HERE
    ```

### Running the App

Start the development server:

```bash
npm start
# or yarn start
